# Sistema de Gerenciamento de Vendas

Este é um projeto desenvolvido para a disciplina de Desenvolvimento Web 2.

## Descrição

Sistema para gerenciamento de vendas com funcionalidades para controle de produtos, vendas e relatórios.

## Tecnologias Utilizadas

- Sistema desenvolvido utilizando tecnologias web modernas
- Gerenciamento de vendas e produtos
- Interface intuitiva e responsiva

## Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/karemoliveira18/dw2--karem-oliveira---vendas-..git
```
2. Instale as dependências
3. Execute o projeto
